Class AudioProcessor

Namespace
Aura.Core.Audio

Provides audio processing utilities including LUFS normalization and DSP chain. Spec requirement: -14 LUFS (YouTube) / -16 LUFS (voice-only) / -12 LUFS (music-forward) DSP chain: HPF -> De-esser -> Compressor -> Limiter
