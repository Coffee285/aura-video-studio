Interface IOllamaDirectClient

Namespace
Aura.Core.Providers
Assembly
Aura.Core.dll

Direct client for Ollama API without going through the LLM provider abstraction. This interface provides a clean, dependency-injection-friendly way to call Ollama without using reflection to access private fields of OllamaLlmProvider.

ARCHITECTURAL DECISION: Use this interface instead of reflection-based access. Reflection is fragile and breaks when provider implementation changes.
