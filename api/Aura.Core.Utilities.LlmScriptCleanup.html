Class LlmScriptCleanup

Namespace
Aura.Core.Utilities
Assembly
Aura.Core.dll

Comprehensive utility for cleaning LLM script output to remove meta-information that should not be included in TTS narration.

This addresses the issue where LLMs add metadata like:

  • Word Count: X
  • TTS Pacing Check: ...
  • AI Detection Avoided: ...
  • Visual Synergy: ...
  • Emotional Flow: ...
  • Accuracy notes

All LLM providers should use this utility to ensure clean script output.
