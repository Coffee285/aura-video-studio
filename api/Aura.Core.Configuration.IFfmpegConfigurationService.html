High-level facade for FFmpeg configuration, combining:

  • FFmpegOptions (appsettings)
  • Environment hints (e.g. AURA_FFMPEG_PATH)
  • Persistent configuration (FFmpegConfigurationStore) This is the single authoritative access point for FFmpeg config at runtime.
