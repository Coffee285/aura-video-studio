Badge Legend

Badge Color & Icon Key:

🔴 Run Override (Pinned)     → Highest precedence, blocks if unavailable
🟠 Run Override              → High precedence, falls back if unavailable
🟡 Stage Pinned              → Per-stage lock, blocks if unavailable
🔵 Project Override          → Project-level preference
⚪ Global Default            → Application-wide default
🟢 Automatic Fallback        → Safe catalog default (if enabled)

🔒 Pinned                    → Will never auto-change
⚠️ Deprecated                → Consider migrating
✓ Used                       → Successfully used in job
⚠️ Blocked                   → Unavailable and requires user action
